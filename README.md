# Projeto: Teste Unitário com Python

Repositório criado para a aula prática de testes unitários utilizando o módulo `unittest` (PyUnit) na linguagem Python.

## Funcionalidades Implementadas

* **Operações Matemáticas Básicas:** Funções para soma, subtração, multiplicação e divisão.
* **Potência:** Função que calcula a base elevada a um expoente.
* **Média de Lista (Desafio Extra):** Função que calcula a média dos números contidos em uma lista. Conta com tratamento de exceção, lançando um erro do tipo `ValueError` caso a lista fornecida esteja vazia.

## Como executar os testes

Para verificar o funcionamento das funções e rodar os testes unitários, abra o terminal na raiz do projeto e execute:

```bash
python -m unittest test_calculadora.py
```
